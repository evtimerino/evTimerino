#### Precision
Changes the precision, this resets the timer
Availables precisions:

<ul>
  <li>1/2</li>
  <li>1/3</li>
  <li>1/4</li>
  <li>1/6</li>
  <li>1/8</li>
  <li>1/12</li>
  <li>1/16</li>
</ul>

#### Test strip mode

**Separate** every steps are single exposures with the time calculated with base exposure plus increments (to be used with localized test strip)
**Split Grade** every steps are two exposures, the base exposure with one filter and the increment with the other filter (to be used with localized test strip)
**Incremental** first step is the base exposure, consecutives are incremental

#### Brightness

Adjust the brightness of the display

#### Safelight

<ul>
  <li>off</li>
  <li>Off on exposure</li>
  <li>Always on</li>
</ul>

#### Buzzer

**On** beeps every seconds after starting the base exposure
**Off** Stay silent during the base exposure

#### Prepare

This is an explorative feature that allow to switch on the lamp before every adjustments to allow the best positioning with hands.
