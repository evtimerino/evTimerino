# QUICK MANUAL

###### TESTSTRIP:

The timer stars with the **separate** test strip mode that requires an external localized test printer.
This is the preferred method since it gives maximum precision

<ol>
  <li>Select the test strip mode</li>
  <li>Cut a strip of paper and place in the test strip printer</li>
  <li>start the timer</li>
  <li>Move the strip of paper to ne next step</li>
  <li>Start again the timer</li>
</ol>

###### EXPOSURE:

<ol>
  <li>Choose the strip in the test strip print with the correct exposure</li>
  <li>Move up and down the main exposure to match the exposure time of the strip</li>
  <li>Start the timer</li>
</ol>

###### ADJUSTMENTS:

<ol>
  <li>Enter in the adjustments mode</li>
  <li>Move up or down to create an adjustment</li>
  <li>Long press adjustment button to save it</li>
  <li>A confirmation box will appear in the left corner of the oled display</li>
  <li>Start the timer</li>
</ol>
